# Shared Agent Root

이 파일은 여러 프로젝트의 `AGENTS.md`와 `.claude/CLAUDE.md`에 symlink로 연결되는 공통 루트 문서입니다.

## 공통 원칙

- 변경은 작고 검증 가능하게 유지합니다.
- 기존 구조와 패턴을 먼저 따르고, 새 추상화는 필요할 때만 도입합니다.
- 동작 변경이 있으면 사용자 흐름 기준으로 영향 범위를 설명합니다.
- 비밀값, 토큰, 운영 데이터를 코드, 문서, 로그에 남기지 않습니다.

## 기본 작업 절차

1. 수정 대상과 인접한 파일을 먼저 읽습니다.
2. 서버/클라이언트 경계, 데이터 흐름, 테스트 범위를 확인합니다.
3. 최소 범위로 수정합니다.
4. 가능한 가장 작은 검증부터 실행합니다.
5. 최종 보고에는 변경 파일, 변경 이유, 실행한 검증, 실행하지 못한 검증을 포함합니다.

## 공통 기준

- 프로젝트 고유 규칙이 있으면 저장소 내부 문서를 우선 확인합니다.
- 관련 없는 리팩터링이나 대규모 포맷 변경은 피합니다.
- 테스트를 실행하지 못했다면 이유를 명확히 남깁니다.
- 이 문서는 shared plugin repo에서 관리되므로, 설치된 프로젝트에서는 직접 수정하지 않습니다.
